# Desktop App

This directory is reserved for the future OpenTypeless desktop application.

## Product target

The desktop app should eventually provide:
- global shortcut driven dictation
- AI-assisted rewrite before insertion
- insertion into the active desktop app
- local history and dictionary
- settings for audio, shortcuts, and privacy

## Current status

No implementation has been committed yet.
The repository is still in product and architecture definition.

## Expected early responsibilities

- shell and lifecycle management
- permission onboarding
- global shortcut handling
- audio capture
- state display
- history and settings UI
