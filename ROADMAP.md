# Roadmap

This roadmap describes the current intended sequence for OpenTypeless.
It is directional, not a promise.

## Phase 0 - Foundation

Goal: define the product clearly before implementation expands.

Focus:
- competitor benchmark research
- desktop-only scope definition
- architecture framing
- repository standards and contribution workflow

## Phase 1 - Desktop MVP

Goal: prove the core desktop workflow end to end.

Target capabilities:
- global shortcut to start dictation
- microphone selection
- audio capture pipeline
- speech-to-text integration
- AI rewrite of raw transcript
- insertion into focused text fields
- local history
- personal dictionary
- basic settings and privacy controls

Success signals:
- works reliably in common desktop text fields
- output is materially better than raw transcription
- system state is visible and predictable

## Phase 2 - Power editing

Goal: move beyond dictation into system-wide voice editing.

Target capabilities:
- selected-text editing
- translation mode
- app-specific style or tone profiles
- correction-driven learning
- better progress and failure feedback
- initial Windows parity work

## Phase 3 - Assistant workflows

Goal: support broader knowledge-work actions while staying desktop-first.

Target capabilities:
- ask about selected text
- optional web lookup mode
- Markdown-aware output where useful
- export and import for history and dictionary
- optional local or hybrid backend routing

## Non-goals for now

- mobile apps
- team billing and org admin
- plugin marketplace
- broad workflow automation platform
