# Support

## Getting help

If you need help with OpenTypeless:

- open a GitHub issue for bugs or clearly scoped requests
- open an issue for product ideas, architecture questions, or early design feedback

## Before opening an issue

Please include:

- what you were trying to do
- what happened instead
- your operating system and environment details
- the smallest reliable reproduction you know

## Security issues

Do not report security issues in a public issue first.
Please follow `SECURITY.md`.
