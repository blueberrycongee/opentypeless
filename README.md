# OpenTypeless

OpenTypeless is a desktop-first open source AI dictation layer inspired by the workflow of Typeless.

The initial focus is desktop only:
- macOS first
- Windows second
- no mobile app in the first phase

## Vision

OpenTypeless should let users speak anywhere on desktop and get clean, structured, ready-to-send text back into the current app.

Core experience goals:
- global shortcut to start dictation
- low-latency audio capture
- speech-to-text plus AI rewrite
- insertion into the active desktop app
- local history, dictionary, and personalization
