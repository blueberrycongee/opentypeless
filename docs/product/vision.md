# Product Vision

## Positioning

OpenTypeless is a desktop-first AI dictation layer for people who want to speak naturally and write everywhere.

## The core problem

Speaking is faster and easier than typing in many situations, but ordinary dictation usually fails at the last mile.
Raw transcripts are often noisy, awkward, and not ready to send.

## The product promise

OpenTypeless should let a user speak into any desktop workflow and receive text that is:
- cleaner than raw transcription
- appropriate for the current task
- easy to trust and revise
- available again from local history

## Product principles

- desktop-first, not mobile-first
- cross-app workflow over standalone editor behavior
- clarity over magic
- privacy-aware by design
- incremental capability layers over one giant black box

## What success looks like

A user should be able to:
- open any common desktop app
- trigger dictation with one shortcut
- speak naturally, including pauses and self-corrections
- receive polished text in the current input field
- recover or reuse the result from history later
