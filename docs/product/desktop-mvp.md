# Desktop MVP

## Product goal

Build a desktop-first open source AI dictation layer that works across desktop apps.

## In scope

- global shortcut to start and stop dictation
- audio capture from selected microphone
- speech-to-text pipeline
- AI cleanup of raw transcript
- insertion into the active desktop app
- local history
- custom dictionary
- basic settings

## Out of scope for MVP

- mobile app
- team billing and admin
- plugin marketplace
- advanced workflow automation

## Platform priority

1. macOS
2. Windows

## Success criteria

- users can dictate into common desktop text fields
- output is cleaner than raw transcription
- the product feels fast and predictable
- history and dictionary are usable from day one
