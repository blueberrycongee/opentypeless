# Typeless Benchmark Notes

## What we are benchmarking

Typeless behaves like a desktop AI dictation layer rather than a standalone note editor.

## Core benchmark areas

- global shortcut invocation
- push-to-talk and hands-free dictation
- AI rewrite after transcription
- insertion into the current app
- local history and recall
- personalization and dictionary
- multi-language support
- selected-text editing workflows

## Desktop-first takeaway

The strongest benchmark for OpenTypeless is not plain speech-to-text.
The benchmark is the full desktop workflow:

capture -> transcribe -> rewrite -> insert -> remember -> improve
